## PBI-001: Landing Page: HwHub プロダクト紹介ページ

**優先度**: 高  
**ステータス**: 未着手  
**作成日**: 2026-04-03

**As a** 家事分担に悩んでいる共働き夫婦・シェアハウス住民・シェアハウス運営者が  
**I want to** HwHubがどんなアプリか・自分に合うかを一目で理解したい  
**So that** 安心してアカウント登録・ログインに進める

### ターゲットユーザー

- **共働き夫婦**: 家事の分担・可視化をしたい
- **シェアハウス住民**: メンバー間で掃除などの家事を分担したい
- **シェアハウス運営者**: 複数世帯・複数ハウスをまとめて管理したい

### Acceptance Criteria

- [ ] AC1: ヘッダーにロゴ・ナビ・「ログイン」「サインアップ」ボタンがある
- [ ] AC2: ファーストビューにキャッチコピーと「サインアップ」CTAボタンがある
- [ ] AC3: 以下の機能を訴求するセクションがある
  - 家族・シェアメンバーで家事を分担・管理できる
  - 家事タスクのテンプレート化・定期実行・担当者割当
  - 買い物リストの共有
  - サポートのレスが早い（AIによる一次対応）
  - PCでもスマホでもブラウザで利用可能（アプリ化予定）
- [ ] AC4: ターゲット別（共働き夫婦・シェアハウス住民・シェアハウス運営者）に使いどころを訴求するセクションがある
- [ ] AC5: フッターに基本的なリンク（利用規約・プライバシーポリシーなど）がある
- [ ] AC6: 認証不要で表示される（ログイン状態に関わらず同一表示）
- [ ] AC7: レスポンシブ対応（PC・スマホ両対応）
- [ ] AC8: 「ログイン」「サインアップ」ボタンは既存の `/login` `/register` に繋がっている

### 備考

- **優先順位の根拠**: HwHubにLanding Pageが存在しないため最優先
- **依存関係**: hw-hub-frontendの既存認証フロー（ログイン・サインアップ）
- **実装**: hw-hub-frontendに新規ページとして追加（認証不要ルート）
