## Issue #44: スケルトンスクリーンを導入したい

**GitHub Issue**: ryokkon624/hw-hub-manage#44
**Story Points**: 3
**ブランチ**: feature/44-skeleton-screen
**対象リポジトリ**: hw-hub-frontend

### ユーザーストーリー

As a アプリをモバイルで使うユーザー
I want to データ読み込み中にコンテンツの枠組みをスケルトンで確認したい
So that 画面の構造を把握しながら待て、体感的な待ち時間を短く感じられる

### 実装方針

- 現在の `v-if="isLoading"` でローディング全体を隠すのではなく、データがない間だけ表示する専用コンポーネントを作る
- `SkeletonItem` 共通コンポーネントを作成し、対象3画面で再利用する
- スケルトン表示中はローディングサークルを非表示にする

### Acceptance Criteria

- [x] AC1: 家事割り当て画面のデータ読み込み中に、各行の形状に合ったスケルトンが表示される
- [x] AC2: My Tasks画面のデータ読み込み中に、スケルトンが表示される
- [x] AC3: 買い物リスト画面のデータ読み込み中に、スケルトンが表示される
- [x] AC4: スケルトン表示中は現在のローディングサークルは非表示になる
- [x] AC5: SkeletonItem 共通コンポーネントが作成され、対象3画面で再利用される
