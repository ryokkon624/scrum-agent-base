## Issue #47: [SP版]家事割り当て画面の家事タスクカードにスワイプを導入したい

**ブランチ**: `feature/47-add-swipe-to-assignment`
**コミット参照**: `(ryokkon624/hw-hub-manage#47)`

### 概要

SP版の家事割り当て画面の家事タスクカードにスワイプを導入したい。
※PC版は現状どおり。

| 操作 | アクション | 背景色 | アイコン | 文字 (JP/EN/ES) |
|--|--|--|--|--|
| 左スワイプ | 自分にする | bg-sky-500 (空色) | UserCheck | 自分にする / Mine / Para mí |
| 右スワイプ | メンバーへ(モーダル起動) | bg-amber-500 (琥珀色) | Users | メンバーへ / Members / Miembros |

スワイプ時の背景はアイコンだけだとわかりづらいため、iOSのメールアプリのように、背景に大きくアイコンを薄く（不透明度10〜20%くらいで）配置し、その上にクッキリとした白文字で DONE や SKIP と表示する。

### モーダルについて

表示：おうちのメンバーのアイコン(なければイニシャル)+名前 + 未割当を並べる。
アクション：メンバーピル、もしくは、未割当ピルをタップしたら割り当て
表示時：スワイプの指を話した瞬間にシュッとしたからせりあがってくるような感じ

右スワイプでモーダルが開く際、スワイプの勢い（速度）をそのままモーダルのアニメーションに繋げるように見せれつろBEST。

### ユーザーストーリー

**As a** 家のメンバー
**I want to** スワイプ操作で家事の担当者を素早く変更したい
**So that** 一覧画面から直感的に割り当てを変えられる

### Acceptance Criteria

- [x] AC1: SP版の家事タスクカードを左スワイプすると「自分にする」が実行される（bg-sky-500 背景、UserCheck アイコン、白文字「自分にする/Mine/Para mí」）
- [x] AC2: SP版の家事タスクカードを右スワイプするとメンバー選択モーダルが起動する（bg-amber-500 背景、Users アイコン、白文字「メンバーへ/Members/Miembros」）
- [x] AC3: スワイプ背景には薄い（不透明度10〜20%）アイコンと、くっきりした白文字が重なって表示される
- [x] AC4: モーダルには家のメンバー（アイコンまたはイニシャル＋名前）と「未割当」が並び、タップで割り当てが変更される
- [x] AC5: モーダルはスワイプの指を離した瞬間に下からせり上がるアニメーションで表示される
- [x] AC6: PC版の表示・操作に影響しない

### 備考

- 依存関係：なし
- 補足：右スワイプの勢い（速度）をモーダルのアニメーションに連動させるとなお良い（BEST要件、必須ではない）

## リスク・チャレンジ

- モーダルアニメーション（下からせり上がる）の実装が複雑。スワイプ速度との連動はBEST要件だが、実装難度が高い
- 家事割り当て画面のスワイプ対象コンポーネントの特定と既存実装との整合性確認が必要
- SP版/PC版の条件分岐（AC6）を確実に担保する必要あり（前スプリントと同パターン）
