## Issue #23: SP版ヘッダーコンパクト化

**ユーザーストーリー**: スマートフォンでHousework Hubを使うユーザーとして、画面タイトルが折り返さずに表示されてほしい。そうすることでヘッダーが見やすく、画面スペースを無駄なく使える。

**概要**:

- LanguageSwitcherをテキスト並びからlanguagesアイコン1つに変更（SP版のみ）
- アイコンタップでハーフモーダル（ボトムシート）を起動し言語選択できるようにする
- タイトルに `clamp()` による動的フォントサイズと `text-overflow: ellipsis` を適用（SP版のみ）

**参考コンポーネント**: `HouseholdSwitcherMobile.vue`（ハーフモーダルの実装イメージ）

**AC一覧**:

- [ ] AC1: SP版で LanguageSwitcher が「JA/EN/ES」テキスト並びから languages アイコン1つに変わっている
- [ ] AC2: languages アイコンタップでハーフモーダル（ボトムシート）が起動し、言語を選択できる
- [ ] AC3: ハーフモーダルで現在選択中の言語がわかるようになっている
- [ ] AC4: SP版で画面タイトルが `clamp()` による動的フォントサイズ調整と `text-overflow: ellipsis` で折り返さない
- [ ] AC5: PC版のレイアウト・動作は変更されていない

**GitHub Issue**: `(ryokkon624/hw-hub-manage#23)`

## Issue #24: ボタン配置の統一

**ユーザーストーリー**: Housework Hubを使うユーザーとして、画面内のボタン配置が統一されてほしい。そうすることで操作の迷いなくアプリを使える。

**概要**:

- 各画面のボタンを以下の方針に統一する
  - 買い物アイテム登録画面: 「追加する」→右側、「キャンセル」→「追加する」の左側
  - 買い物アイテム編集画面（PC版・SP版）: 「戻る」ボタン→画面上部左の「< キャンセル」リンク（問い合わせ詳細画面同様）
  - ホーム画面「おうちに参加・参加する」カード: ボタン→カード内下部右
  - ホーム画面「家事テンプレートを作成する」カード: ボタン→カード内下部右

**AC一覧**:

- [x] AC1: 買い物アイテム登録画面の「追加する」ボタンが画面下部右側に配置されている
- [x] AC2: 買い物アイテム登録画面の「キャンセル」ボタンが「追加する」ボタンの左側に配置されている
- [x] AC3: 買い物アイテム編集画面（PC版・SP版）の「戻る」ボタンが画面上部左の「< キャンセル」リンクに変わっている
- [x] AC4: ホーム画面「おうちに参加・参加する」カードのボタンがカード内下部右に配置されている
- [x] AC5: ホーム画面「家事テンプレートを作成する」カードのボタンがカード内下部右に配置されている

**GitHub Issue**: `(ryokkon624/hw-hub-manage#24)`
