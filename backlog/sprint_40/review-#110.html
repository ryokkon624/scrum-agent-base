原因分析

🔍 nicknameを無視してdisplayNameをハードコードで参照していた

member_summary_strip.dart の members.map() 内で、各メンバーのラベルを label: m.displayName とハードコードしていた。APIレスポンスでnicknameが設定されていても、displayNameが常に優先されて表示される実装になっていた。

フィールド 修正前の実装（誤） 修正後の実装（正）
表示名 m.displayName（常に） → nickname != null && isNotEmpty ? nickname : displayName

修正前: member_summary_strip.dart の members.map((m) { return _SummaryChip(label: m.displayName, ...); }) がnicknameを一切参照していなかった。
