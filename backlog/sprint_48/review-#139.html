AC1
GET /api/inquiries でログインユーザーの問い合わせ一覧が表示される

InquiryRepositoryImpl.fetchInquiries() が GET /api/inquiries を呼び出し、レスポンスの {"items": [...]} ラッパー形式からリストを取得する。InquiryListNotifier.build() の初期化時に Future.microtask で非同期ロードを開始する。

lib/features/inquiry/data/inquiry_repository.dart 新規
33+ Future<List<InquirySummaryDto>> fetchInquiries() async {
34+ try {
35+ final response = await _dio.get<Map<String, dynamic>>('/api/inquiries');
36+ final data = response.data!;
37+ return (data['items'] as List<dynamic>)
38+ .map((e) => InquirySummaryDto.fromJson(e as Map<String, dynamic>))
39+ .toList();
40+ } on DioException catch (e) {
41+ if (e.error is AppException) throw e.error!;
42+ throw NetworkException(e.message ?? 'Network error');
43+ }
44+ }
💡 Dio の型パラメータに Map<String, dynamic> を明示。<dynamic> のままにすると2段階キャストが必要になるため規約通り具体型を指定した。
lib/features/inquiry/presentation/inquiry_list/inquiry_list_notifier.dart 新規

💡 AutoDisposeNotifier を採用。画面離脱後にプロバイダーを自動破棄し、次回アクセス時は常に最新データを取得する設計。
