## Issue #87: [mobile] 家事分担画面を実装する (#11)

### ユーザーストーリー

**As a** HwHubモバイルアプリのユーザー
**I want to** 家事タスクの担当者をスワイプで素早く割り当てたい
**So that** 隙間時間に家事の担当調整を手軽にこなせる

### Acceptance Criteria

- [x] AC1: メンバー別未対応件数サマリを横スクロールで表示する。「未割当」を先頭に固定し、ログイン中ユーザーをハイライト表示する
- [x] AC2: フィルタ（すべて / 未割当のみ / 自分+未割当）でフロント側絞り込みができる
- [x] AC3: タスクカードに担当バッジ・家事名・実施日を表示する
- [x] AC4: 左スワイプ（sky-500背景・「自分にする」ラベル）でタスクを自分に割り当てる（`PATCH /api/housework-tasks/{taskId}/assign`）
- [x] AC5: 右スワイプ（amber-500背景・「メンバー選択」ラベル）でメンバー選択モーダルを起動し、任意のメンバーへ割り当て（未割当に戻すことも可）
- [x] AC6: 過去の未割当タスクが存在する場合のみ「過去の未割当をスキップ」ボタンを表示し、確認ダイアログ後に一括スキップできる（`PATCH /api/housework-tasks/bulk-status`）
- [x] AC7: [未割当を自分に割り当てる ▶] / [他メンバーのタスクを奪う ▶] ボタンからスワイプモードを起動できる。0件の場合はグレーアウト（非活性）
- [x] AC8: スワイプモードではカードを1枚ずつ表示し、左スワイプで自分に割り当て・右スワイプで次へ進む。画面上部に進捗（N / 全件）を表示する
- [x] AC9: スワイプモードで全カードを消化した場合、完了メッセージを表示して通常リストモードに戻り最新データを再取得する
- [x] AC10: スワイプ操作はカードがドラッグに追従し、カード幅30%超えでアクション確定
- [x] AC11: 複数世帯所属時は世帯インジケーターバーを表示し、タップで世帯切り替えができる（HouseholdSwitcherはホームで実装済みのUIを再利用）
- [x] AC12: 画面表示時・世帯切り替え時にAPIからデータを取得し、ローディング中はインジケーターを表示する
- [x] AC13: テストカバレッジ ≥95%（Notifier・ウィジェットテスト）

### 画面構成

#### 通常リストモード

```
┌─────────────────────────────────────┐
│ 家事割り当て                          │
├─────────────────────────────────────┤
│ メンバー別未対応件数サマリ             │
│ [未:40] [お兄:0] [父:5] [母:8] ...   │
├─────────────────────────────────────┤
│ 未対応タスク 56件（未割当: 40件）      │
│ 過去の未割当: 31件                    │
│ [過去の未割当をスキップ]               │
├─────────────────────────────────────┤
│ 表示対象: [すべて] [未割当] [自分+未] │
├─────────────────────────────────────┤
│ タスクカード一覧（スワイプ操作可）     │
├─────────────────────────────────────┤
│ [未割当を自分に割り当てる ▶]          │
│ [他メンバーのタスクを奪う ▶]          │
└─────────────────────────────────────┘
```

#### スワイプモード（Tinder風）

```
┌─────────────────────────────────────┐
│ ← 自分に割り当てる          3 / 40  │
├─────────────────────────────────────┤
│  ┌───────────────────────────────┐  │
│  │  猫砂補充                     │  │
│  │  実施日: 2026-04-20           │  │
│  │  現在の担当: 未割当            │  │
│  └───────────────────────────────┘  │
│   ← 自分にする        次へ →        │
└─────────────────────────────────────┘
```

### API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/housework-tasks` | タスク全件取得。フィルタはフロント側で実施 |
| GET | `/api/households/{householdId}/members` | 世帯メンバー一覧取得（サマリ表示用） |
| PATCH | `/api/housework-tasks/{taskId}/assign` | 担当者変更（左スワイプ自己割り当て / メンバー選択モーダル / スワイプモード割り当て共通） |
| PATCH | `/api/housework-tasks/bulk-status` | 過去の未割当を一括スキップ |

### 状態管理

```
HouseworkAssignState
├── tasks: List<HouseworkTask>         // タスク一覧
├── members: List<HouseholdMember>     // 世帯メンバー（サマリ用）
├── filter: AssignFilter               // すべて / 未割当 / 自分+未割当
├── mode: AssignMode                   // list / swipe
├── swipeTarget: SwipeTarget           // unassigned / others
├── swipeIndex: int                    // スワイプモードの現在位置
└── isLoading: bool
```

### スワイプ仕様

**通常リストモード**:

| 操作 | 背景色 | ラベル |
|---|---|---|
| 左スワイプ（自分に割り当て） | sky-500（青）| 自分にする |
| 右スワイプ（メンバー選択） | amber-500（橙）| メンバー選択 |

**スワイプモード**:

| 操作 | 背景色 | ラベル |
|---|---|---|
| 左スワイプ（自分に割り当て） | sky-500（青）| 自分にする |
| 右スワイプ（次へ） | グレー（slate-400）| 次へ |

- カード幅30%超えでアクション確定
- 終了後は通常リストを最新状態に再取得

### フィルタ処理（フロント側）

| フィルタ | 絞り込み条件 |
|---|---|
| すべて | 絞り込みなし |
| 未割当のみ | `assignee == null` |
| 自分＋未割当 | `assignee == me || assignee == null` |
