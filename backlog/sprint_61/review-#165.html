原因分析

🔍 _swipeTargetTasksFor() の others フィルタに「自分除外」条件がなかった

_swipeTargetTasksFor() の SwipeTarget.others ブランチは「担当者ありのタスク全て」を返していた。担当者ありには自分のタスクも含まれるため、「他メンバーのタスクを奪う」モードで自身のタスクも表示されてしまっていた。

SwipeTarget 修正前のフィルタ条件 修正後のフィルタ条件
unassigned assigneeUserId == null（変更なし） assigneeUserId == null
others assigneeUserId != null
（自分のタスクも含む）		 assigneeUserId != null
&& assigneeUserId != currentUserId
（自分を除外）

同じフィルタロジックが3箇所に存在しており、全箇所の修正が必要だった：

  1. _swipeTargetTasksFor()（Notifier 共通ヘルパー）
  2. _ListModePage.build()（スワイプボタンの有効化判定・件数表示）
  3. _SwipeModePage.build()（スワイプ中の現在カード取得）

実装メモ（仕様外の判断・変更・妥協点）

シグネチャ変更 startSwipeMode(SwipeTarget) → startSwipeMode(SwipeTarget, int currentUserId) に変更。Page から currentUserId を渡す設計にすることで、Notifier が authNotifierProvider に依存する箇所を1箇所（swipeAssignToMe 内）に抑えた。
swipeAssignToMe の取得方法 swipeAssignToMe 内での currentUserId は authNotifierProvider.future から毎回取得する形を採用（シグネチャには含めない）。スワイプ中に currentUserId は変化しない前提のため問題なし。既存の assignToMe と一貫したパターン。
既存テスト修正 startSwipeMode(others) の既存テストケースは currentUserId=10 でタスクの assignee=10（自分のみ）という構成だったため、修正後は swipeTaskCount=0 になった。タスク id=3（assignee=20、他人）を追加してテスト意図を明確化した。

AC確認

✓
AC1
「他メンバーのタスクを奪う」スワイプモードに自身のタスクが表示されない
_swipeTargetTasksFor() の others ブランチに && t.assigneeUserId != currentUserId を追加し、自身のタスクを除外。Notifier・Page（リストモード・スワイプモード）の3箇所全てに修正を適用した。
✓
AC2
「他メンバーのタスクを奪う」スワイプモードに未割当タスクと自身を除く他メンバーのタスクが表示される
  • 未割当タスク（assigneeUserId == null）: 「未割当タスクを自分にアサイン」モード（SwipeTarget.unassigned）で表示。others モードには表示されない
  • 他メンバーのタスク（assigneeUserId != null && != currentUserId）: 「他メンバーのタスクを奪う」モード（SwipeTarget.others）で表示
  • 自分のタスク（assigneeUserId == currentUserId）: どちらのモードにも表示されない

動作確認ポイント（ユーザー確認用）

  • 1
    「他メンバーのタスクを奪う」ボタンを押したとき、自分のタスクが表示されないこと 自分に割り当て済みのタスクがある状態で「他メンバーのタスクを奪う」スワイプモードに入り、自分のタスクがリストに出ないことを確認
  • 2
    「他メンバーのタスクを奪う」ボタンが、他メンバーのタスクが1件もない場合に無効化されること 全タスクが自分のタスクまたは未割当だけの場合、「他メンバーのタスクを奪う」ボタンがグレーアウトしていることを確認
  • 3
    「未割当タスクを自分にアサイン」スワイプモードは引き続き正常動作すること 未割当タスクが正しく表示され、スワイプ操作で自分に割り当てられることを確認（今回の修正の影響範囲外だが、デグレ確認）
  • 4
    他メンバーのタスクを奪うスワイプを完了したとき、リストモードに正常に戻ること 全カードを消化した後、「完了」表示になり、リロード後にリストモードに戻ることを確認
